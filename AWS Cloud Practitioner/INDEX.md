# AWS Cloud Practitioner

## Quick Summary

AWS Cloud Practitioner is the foundation-level AWS path. It focuses on cloud concepts, AWS global infrastructure, core services, security, monitoring, pricing, support, and shared responsibility.

## Learning Order

| Order | Note | Focus |
| --- | --- | --- |
| 1 | [Cloud Foundation and Compute](1%20-%20Cloud%20Foundation%20and%20Compute.md) | Cloud basics, service/deployment models, global infrastructure, Well-Architected, accounts, IAM basics, EC2, compute options. |
| 2 | [Storage, Networking and Database](2%20-%20Storage%2C%20Networking%20and%20Database.md) | EBS, EFS, S3, FSx, backup, CloudFront, Global Accelerator, VPC, Route 53, databases. |

| 5 | [AWS Terminology Cheat Sheet](5%20-%20AWS%20Terminology%20Cheat%20Sheet.md) | Fast revision terms. |

## Exam Orientation

Checked against AWS CLF-C02 official exam guide on 2026-05-18. Re-check the official guide before booking because AWS exam guides and service emphasis can change.

Main knowledge areas:

- Cloud concepts.
- Security and compliance.
- Cloud technology and services.
- Billing, pricing, and support.

## Revision Method

1. Learn the service category.
2. Learn what the service is used for.
3. Learn when not to use it.
4. Learn shared responsibility boundaries.
5. Learn pricing model basics.
6. Practice explaining each service in one sentence.
