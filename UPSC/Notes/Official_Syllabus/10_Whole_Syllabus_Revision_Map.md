## Quick Summary

- This file converts the official syllabus into a revision sequence.
- Use it to plan weekly study without losing sight of the whole syllabus.
- It separates base reading, expansion, current-affairs linkage, answer writing, and test practice.

## Weekly Template

| Day | Core task | Output |
| --- | --- | --- |
| Monday | Static subject revision | 2 pages of crisp notes or 1 chapter revision |
| Tuesday | Current affairs mapping | 5 events mapped to static syllabus |
| Wednesday | Prelims practice | 50 MCQs plus error log |
| Thursday | Mains answer practice | 2 answers from current static topic |
| Friday | Gap topic expansion | 1 missing syllabus note |
| Saturday | CSAT practice | 1 timed set |
| Sunday | Weekly audit | Update coverage map and revise mistakes |

## Subject Revision Blocks

## Topic: Polity

## Meaning:
- Constitution, institutions, governance, rights, federalism, democracy.

## Revision sequence:
- Existing NCERT notes.
- Article/body/function notes.
- Current issues.
- Mains answers.

## Topic: History

## Meaning:
- Ancient, medieval, modern, national movement, world, post-independence.

## Revision sequence:
- Existing modern-history NCERT notes.
- Add missing ancient/medieval/culture.
- Timeline sheets.
- Movement/personality questions.

## Topic: Geography

## Definition:
- Physical, Indian, world, resources, industries, society-environment link.

## Map location:
- Practise daily map marking: rivers, passes, ports, minerals, industries, national parks, world straits.

## Diagram needed?
- Yes: monsoon, cyclones, earthquakes, river basin, soil, plate tectonics.

## Concept: Economy

## Meaning:
- Growth, development, employment, budget, agriculture, money, banking, external sector.

## Why important:
- Economy connects static concepts with current data.

## Government/RBI link:
- Budget, Economic Survey, RBI policy, schemes, financial regulators.

## Topic: Environment

## Meaning:
- Ecology, biodiversity, climate, pollution, conservation, law, institutions.

## Current affairs link:
- Species, reports, conventions, disasters, climate policy, pollution orders.

## Topic: Ethics

## Meaning:
- Values, integrity, aptitude, public service, emotional intelligence, probity, case studies.

## Revision sequence:
- Definitions.
- Examples.
- Thinkers.
- Case frameworks.
- Timed writing.

## Prelims Pointers

- Keep an error log by subject, not by test only.
- Mark every wrong answer as concept gap, fact gap, current-affairs gap, or careless error.
- Revise maps, bodies, reports, schemes, and environment facts repeatedly.

## Mains Pointers

- Every static topic needs: definition, constitutional/legal link if any, current example, criticism, reform, conclusion.
- Practise answer intros and conclusions separately.
- Maintain a small value-add bank instead of collecting endless data.

## Keywords for Answer Writing

- Revision cycle
- Error log
- Static-current integration
- Answer structure
- Test feedback
- Value addition

## Possible UPSC Questions

- Planning question: How should I revise the full UPSC syllabus without neglecting CSAT and Ethics?
- Planning question: Which current-affairs items should be converted into Mains examples?

## Revision Summary

- Revise in cycles, not once.
- Protect weekly CSAT time.
- Track gaps explicitly.
- Convert current affairs into static notes.
- Do answer writing from the start.
- Update the coverage map after each note batch.
