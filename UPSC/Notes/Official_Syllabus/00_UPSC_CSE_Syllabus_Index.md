## Quick Summary

- This folder is the official-syllabus navigation layer for the UPSC Civil Services Examination.
- It uses the UPSC notification as the source anchor and connects each syllabus area to the NCERT-backed notes already present in `UPSC/Notes`.
- The existing NCERT notes are strong for Polity, Modern History basics, Geography foundations, Economy basics, and Environment foundations.
- The official syllabus is wider than the NCERT batch. The new General Studies layer now gives first-pass coverage for those wider common-paper areas; optional-specific detail still requires the chosen optional subject.

## How To Use These Notes

1. Read this index once to understand the complete exam structure.
2. Use `09_Syllabus_To_NCERT_Coverage_Map.md` to know what is already covered by the NCERT chapter notes.
3. Use the paper-wise notes below for revision buckets, mains dimensions, and missing-topic tracking.
4. Keep current affairs in `UPSC/Monthly GK/` and connect each event to a static syllabus line.
5. Do not treat this folder as a substitute for the full source notes; use it as a syllabus control file.

## Exam Structure

| Stage | Papers | Main purpose |
| --- | --- | --- |
| Prelims | GS Paper I and CSAT Paper II | Screening for Mains |
| Mains qualifying papers | Indian language and English | Qualifying language ability |
| Mains merit papers | Essay, GS I, GS II, GS III, GS IV, Optional I, Optional II | Written merit |
| Personality Test | Interview | Judgement, awareness, balance, and public-service suitability |

## High-Yield Reading Order

1. Polity and Constitution: start with existing Class 11 Polity notes, then expand GS II topics.
2. Modern History and National Movement: use Class 8 and Class 10 History notes, then add missing spectrum-level chronology.
3. Geography: revise NCERT Geography notes, then add world physical geography and mapping.
4. Economy: revise Class 10-12 Economy notes, then add budget, agriculture, infrastructure, and inclusive growth.
5. Environment: revise Biology ecology notes, then add laws, institutions, EIA, climate agreements, species, and protected areas.
6. Ethics: build from dedicated GS IV note and case-study practice.
7. Current affairs: update monthly, mapped to static syllabus.
8. CSAT: practise weekly; do not postpone because it is qualifying.
