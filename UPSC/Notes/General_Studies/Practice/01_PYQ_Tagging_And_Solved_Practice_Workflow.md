## Quick Summary

- Previous-year questions convert notes into exam readiness.
- Every major note should eventually have PYQ tags, repeated themes, and solved answer skeletons.
- Use the official UPSC previous-question page as the primary source for papers.

## Workflow

1. Download official question paper.
2. Split questions by syllabus area.
3. Link each question to existing note.
4. If no note exists, create a gap item.
5. Write answer skeleton, not a polished essay first.
6. Add examples, diagrams, data, and keywords.
7. Review after one week.
8. Rewrite weak answers under time limit.

## Prelims Workflow

## For each wrong question:
- Identify whether the error was concept, fact, elimination, overthinking, or time pressure.
- Add the correction to the relevant note.
- If the question depends on current affairs, add the source and date.

## Mains Workflow

## For each question:
- Identify command word: discuss, analyse, examine, evaluate, comment, critically discuss.
- Define key term.
- Build dimensions.
- Add example/data/case.
- Add criticism or limitation.
- Conclude with balanced way forward.

## Review Rubric

| Criterion | Check |
| --- | --- |
| Relevance | Every paragraph answers the question |
| Structure | Introduction, body, conclusion are clear |
| Content | Static plus current plus example |
| Analysis | Causes, impacts, issues, reforms |
| Balance | Includes limitations/counterview where needed |
| Expression | Direct, simple, no filler |
| Time | Written within target time |

## Prelims Pointers

- Do not only read explanations; record why your option was wrong.
- Use official question papers for pattern familiarity.

## Mains Pointers

- PYQ tagging prevents over-reading.
- Repeated themes deserve one-page answer frameworks.
- Answer writing should start before finishing every source.
