## Quick Summary

- Current affairs is not a separate newspaper pile; it is the annual update layer for every static topic.
- Prioritize official sources for schemes, reports, Budget, Economic Survey, laws, institutions, international agreements, environment, science, security, and welfare.
- Convert current events into syllabus tags, not random notes.

## Source Hierarchy

| Source type | Examples | Use |
| --- | --- | --- |
| Official primary | UPSC, PIB, ministries, Budget, Economic Survey, RBI, laws, Parliament, MEA, MHA, MoEFCC | Facts, schemes, data, reports |
| Official secondary | Annual reports, dashboards, committee reports | Sectoral updates |
| Standard reference | NCERT, government textbooks, official manuals | Static base |
| Newspapers/magazines | Reputed editorial sources | Analysis and debate after fact verification |

## Current Affairs Tagging System

## Required tags:
- Prelims fact.
- GS I history/society/geography.
- GS II polity/governance/social justice/IR.
- GS III economy/science/environment/security/disaster.
- GS IV ethics/case study.
- Essay value-add.
- Map/location.
- Report/data.
- Scheme/law/body.

```text

```

## Monthly Workflow

1. Collect official source links.
2. Tag each item to syllabus.
3. Reject low-value political noise unless it affects policy, law, institution, society, economy, security, or IR.
4. Add static cross-link to the relevant note.
5. Create Prelims fact cards only for facts likely to be asked.
6. Add Mains value only where it improves an answer.
7. Review at month-end and delete duplicates.

## Prelims Revision Method

1. Revise static notes first.
2. Add current facts as annotations.
3. Make short MCQ-style fact cards.
4. Solve previous-year papers.
5. Revisit wrong questions by syllabus topic.
6. Use maps for location-based current affairs.

## Mains Integration Method

## For each current issue, prepare:
- Definition.
- Background.
- Stakeholders.
- Constitutional/legal/institutional link.
- Data point.
- Benefits.
- Challenges.
- Way forward.
- Ethical dimension if relevant.

## What Not To Do

- Do not copy long articles into notes.
- Do not collect unverified figures.
- Do not memorize every announcement.
- Do not detach current affairs from syllabus.
- Do not use current examples without date and source.

## Prelims Pointers

- Focus on schemes, bodies, species, locations, reports, indices, laws, technologies, and institutions.
- Track official names carefully.
- Keep map-based current affairs separate.

## Mains Pointers

- Current examples should support analysis, not replace it.
- Prefer one verified example over many vague examples.
- Update figures close to the exam.

## Keywords for Answer Writing

- Current linkage
- Official source
- Syllabus tagging
- Static-dynamic integration
- Evidence-backed answer
- Data point
- Case example
- Policy update

## Possible UPSC Questions

- How should a current issue be converted into a GS II answer point?
- Which current events are worth tracking for Prelims?
- How can Budget and Economic Survey improve GS III answers?

## Revision Summary

- Current affairs is the update layer for the whole syllabus.
- Every current note needs a source, syllabus tag, static link, Prelims fact, and Mains use.
- Official sources should anchor facts; analysis can be added after verification.
