# Introduction to DevSecOps for Cloud

## Quick Summary

DevSecOps means building security into the software delivery lifecycle instead of treating it as a final approval step. In cloud environments, this includes secure design, secure coding, CI/CD scanning, infrastructure security, secrets handling, deployment controls, monitoring, and incident response.
