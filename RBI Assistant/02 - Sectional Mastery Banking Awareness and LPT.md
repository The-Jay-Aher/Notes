# 02 - Sectional Mastery, Banking Awareness, and LPT

## Why This Chapter Matters

RBI Assistant success depends on mastering ordinary-looking sections under time pressure. English, Numerical Ability, Reasoning, GA, Banking Awareness, Computer Knowledge, and LPT each fail candidates for different reasons. The preparation has to be sectional, timed, and evidence-based.

## The Big Picture

```text
section concept
  -> question pattern
  -> timed drill
  -> accuracy tracking
  -> weak-area repair
  -> mock performance
```

## Section Strategy

### English

Core areas:

- reading comprehension
- cloze test
- error spotting
- sentence improvement
- para jumbles
- vocabulary in context

Method:

Read for meaning first, grammar second. Most errors come from rushing without identifying the sentence structure.

### Numerical Ability

Core areas:

- simplification/approximation
- number series
- arithmetic
- data interpretation
- percentage, ratio, average, profit/loss, SI/CI, time/work, time/speed/distance

Method:

Build formula memory through problem types, not isolated formulas.

### Reasoning

Core areas:

- puzzles and seating
- syllogism
- inequality
- coding-decoding
- blood relation
- direction sense
- alphanumeric series
- input-output where applicable

Method:

Do not start with the longest puzzle if easier marks are available.

### General and Banking Awareness

Core areas:

- RBI functions
- monetary policy basics
- banking products and terms
- payment systems
- financial inclusion
- recent appointments/reports/schemes
- budget/economic survey highlights where relevant

Method:

Maintain a daily/weekly revision table. GA decays fast without spaced revision.

### Computer Knowledge

Core areas:

- hardware/software
- operating system basics
- MS Office basics
- internet and networking
- database basics
- cybersecurity basics
- shortcuts and common terms

Method:

Make one-page tables. Computer knowledge rewards clean factual recall.

### LPT

If applicable, prepare:

- reading local language text
- writing simple official/common sentences
- speaking and comprehension
- local language script confidence

## Small Details That Matter Later

- In reasoning, puzzle selection is a scoring decision.
- In quant, approximation can save time but must not break accuracy.
- In English, context beats memorized word lists.
- GA should include RBI and banking basics, not only headlines.
- Computer section can be a rank stabilizer.
- LPT preparation should begin early if the language is weak.
- Verify local language requirements from the official notification.

## Common Mistakes

| Mistake | Fix |
| --- | --- |
| Solving questions in fixed order | Use easy-to-hard section strategy. |
| Ignoring accuracy | Track wrong attempts by type. |
| Memorizing GA once | Revise with spaced repetition. |
| Avoiding English reading | Practice short daily passages. |
| Leaving LPT for later | Practice basic reading/writing early. |

## Practice Questions

1. Build a 30-day GA revision tracker.
2. Make a list of 50 banking terms with one-line meanings.
3. Solve 10 simplification sets under timer.
4. Attempt 5 reasoning puzzles and classify by difficulty.
5. Write 10 simple sentences in the local language required for the applied office.

## Chapter Summary

RBI Assistant is a precision exam. The winning method is not endless material collection; it is sectional concept clarity, timed practice, accuracy tracking, GA revision, computer recall, and LPT readiness.
