# 10 - MPSC Cheatsheet

## Purpose

Fast revision sheet for MPSC state-services preparation. Verify exact exam pattern and syllabus from the latest official MPSC advertisement/PDF before final planning.

## One-Line Mental Model

```text
UPSC static foundation + Maharashtra-specific history/geography/economy/administration + current affairs + answer writing
```

## Must-Verify Official Items

- exam name and advertisement number
- year and stage pattern
- Prelims papers and negative marking
- Mains papers and marks
- language papers and qualifying rules
- optional subject rules if applicable
- interview/personality test
- eligibility and age limits
- application dates and documents

## Maharashtra-Specific Study Grid

| Area | What to know |
| --- | --- |
| History | Maratha power, colonial Maharashtra, nationalism, social reform, Samyukta Maharashtra. |
| Social reform | Phule, Savitribai, Shahu Maharaj, Ambedkar, reform institutions, education, anti-caste movements. |
| Geography | regions, rivers, rainfall, soils, crops, coast, forests, minerals, drought zones. |
| Economy | agriculture, cooperatives, industry, services, Mumbai/Pune economy, regional imbalance. |
| Polity/admin | state government, district administration, local bodies, constitutional provisions. |
| Current affairs | state schemes, budget, disasters, agriculture, infrastructure, social justice, environment. |

## Answer Framework

```text
context
  -> Maharashtra-specific cause
  -> region/data/example
  -> administrative mechanism
  -> challenges
  -> way forward
```

## Small Details That Matter Later

- Do not confuse MPSC Rajyaseva with every MPSC exam.
- State examples strengthen generic GS answers.
- District/region associations are MCQ traps.
- Scheme details change; verify from official state sources.
- Drought questions need geography plus policy.
- Cooperative questions need economic benefit plus governance challenge.
- Local governance answers need constitutional and state-level implementation.
- Maharashtra history should be studied through causes, institutions, and movements, not only personalities.
